# Counter
Benvenuto nella Counter App! Questa è una semplice applicazione web che permette all'utente di incrementare e decrementare un contatore attraverso l'interfaccia grafica.

## Funzionalità

- Visualizzazione del contatore iniziale a 0.
- Pulsanti per incrementare e decrementare il valore del contatore.
- Stile accattivante e moderno.

## Tecnologie Utilizzate

- HTML
- CSS
- JavaScript
